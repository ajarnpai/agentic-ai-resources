# Agentic AI Resources

Professional landing page and installation guide for the **NotebookLM MCP CLI**, celebrating the **88th Anniversary of Chulalongkorn Business School (CBS)**.

## 🚀 Deployment Instructions

This project is designed to be hosted on **GitHub Pages**.

1. Your site will be live at `https://ajarnpai.github.io/agentic-ai-resources/`.
2. Go to **Settings > Pages** on your GitHub repository.
3. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Click **Save**.

## 📁 Project Structure

- `index.html`: The main landing page (Mobile-responsive, CBS White Theme).
- `guide/`: Detailed technical installation guides.
- `logo/`: CBS 88th Anniversary branding assets.

## ✨ Features

- **Ultra-Beginner Friendly:** Step-by-step guidance for Python and Terminal setup.
- **Premium Aesthetics:** Royal Blue and Deep Maroon CBS branding.
- **One-Click Commands:** Easy copy-to-clipboard for all CLI setup steps.
- **Grounded Intelligence:** Explains the power of connecting Google NotebookLM to Claude, Cursor, and other AI tools.
